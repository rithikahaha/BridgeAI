# BridgeAI — Academic to Industry Skill Gap Analyzer

Every CS graduate has felt it — you finish four years of college and then realise 
the industry wants skills your curriculum never taught you. BridgeAI was built to 
close that gap.

Select your university curriculum and target role, and BridgeAI gives you an honest, 
no-fluff analysis of where you stand — what you already know, what you're missing, 
and exactly what to do about it in the next 90 days.

## What It Does
- Compares your academic curriculum against real industry job requirements
- Identifies critical skill gaps with priority levels (Critical, Important, Recommended)
- Gives you a readiness score out of 100
- Generates a personalised 90-day action plan to close the gaps
- Suggests 3 specific projects to build to become job-ready

## Supported Curriculums
KTU (Kerala), Anna University, VTU (Karnataka), Mumbai University, IT General  
(Don't see yours? Just paste your subjects manually)

## Supported Roles
Data Analyst, Software Engineer, ML Engineer, SAP Consultant, Business Analyst

## Built With
Python, Google Gemini 2.5, LangChain, Gradio

## How to Run
1. Clone this repo
2. Run `pip install -r requirements.txt`
3. Add your Gemini API key in `bridgeai.py`
4. Run `python bridgeai.py`

## Why I Built This
As a CS graduate myself, I experienced this gap firsthand. BridgeAI is my attempt 
to give every student a clear, honest roadmap from academia to industry.
